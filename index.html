A.G.N.E.S.S.

A.G.N.E.S.S. (Ai Gamer usiNg Evolutionary Strategy Searching) is a code that learns how to play any turn-based deterministic, non-random, zero-sum two player game. This is done by implementing a Monte Carlo Tree Search with hyperparameters that are found by an evolutionary algorithm. The cost function is calculated by letting the organisms (sets of hyperparameters) have a smaller tournament were the ranking is decided using the page rank algorithm.

How To Use

To run AGNESS one must first write a rules file. The file called tictactoeRules.py is a template for the rules files.

Next run index.py. When the script finished running it will print the metaparameters used to define a player. These can then be used to create a montePlayer (see index.py for reference) that can play your game.
